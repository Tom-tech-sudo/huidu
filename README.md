# 图像模糊区域检测与灰度补偿

## 项目简介

本项目实现了一套针对黄色晶格图像的自动处理流程，主要功能包括：

1. 自动提取黄色晶格区域；
2. 检测晶格中的模糊区域，并用红色框进行标注；
3. 将检测结果压缩至 **64×36** 像素，作为模糊区域参考图；
4. 将参考图中的模糊区域映射回原始灰度图，并按照**中心补偿高、边缘补偿低**的原则进行灰度补偿。

整个流程适用于黄色晶格图像的模糊区域定位及灰度补偿处理。

# 使用流程

## 第一步：提取黄色晶格区域并检测模糊区域

运行程序：

```bash
python crop.py
```

### 参数说明

修改 `crop.py` 中的以下参数：

```python
input_img = "黄色晶格图片路径"
output_img = "输出图片路径"
```

其中：

* **input_img**：待处理的黄色晶格图片；
* **output_img**：处理结果保存路径。

### 功能说明

程序将自动完成以下操作：

* 读取黄色晶格图片；
* 提取黄色晶格区域；
* 检测晶格中的模糊区域；
* 使用**红色矩形框**标记模糊区域；
* 保存处理后的图片。

### 输出结果

输出图片包含：

* 裁剪后的黄色晶格区域；
* 标记完成的模糊区域（红色框）。

---

## 第二步：图像压缩

运行程序：

```bash
python resize.py
```

### 参数说明

修改：

```python
input_img = "第一步输出图片"
output_img = "压缩后图片输出路径"
```

其中：

* **input_img**：第一步生成的图片；
* **output_img**：压缩后的图片保存路径。

### 功能说明

将第一步得到的图片压缩至：

```text
64 × 36 像素
```

压缩过程中保留模糊区域标记信息，为后续灰度补偿提供参考。

---

## 第三步：灰度补偿

运行程序：

```bash
python blurdivide.py
```

### 参数说明

修改以下参数：

```python
mask_reference_img = "第二步输出图片"
original_bg = "上一步灰度补偿的纯净灰度图"
output_result = "最终输出路径"
```

其中：

* **mask_reference_img**：第二步生成的 64×36 像素参考图片；
* **original_bg**：上一步灰度补偿的纯净灰度图片；
* **output_result**：最终灰度补偿结果保存路径。

### 功能说明

程序将完成以下步骤：

1. 读取 64×36 的参考图片；
2. 提取其中红色框所标记的模糊区域；
3. 将模糊区域映射回原始灰度图片；
4. 对对应区域进行灰度补偿。

补偿策略如下：

* 模糊区域中心补偿幅度较高；
* 越靠近边缘补偿越弱；
* 补偿值平滑过渡，避免出现明显边界。

### 输出结果

输出完成灰度补偿后的灰度图片。

# 注意事项

1. 请确保运行程序前正确配置各脚本中的输入、输出路径。
2. 三个脚本需按以下顺序依次执行：

```text
crop.py
   ↓
resize.py
   ↓
blurdivide.py
```

3. 每一步生成的输出图片均作为下一步程序的输入，请勿随意修改图片尺寸或格式。
4. 建议保持原始灰度图与黄色晶格图具有相同的分辨率，以保证映射精度。

---

# 最终输出

运行完整流程后，可获得一张经过灰度补偿处理的灰度图片。补偿区域由黄色晶格图中的模糊区域自动确定，并依据**中心补偿高、边缘补偿低**的策略完成灰度值调整，从而实现模糊区域的自适应灰度补偿。
